The goal of gpx2pdf is to create pdf files from annotated gpx files

**N.B.**: This is an early draft of a package, unlikely to be of much
use at this stage.

## Installation

You can install `gpx2pdf` with:

``` r
remotes::install_github("giocomai/gpx2pdf")
```
